class regmod.models.Model(data: regmod.data.Data, params: Optional[List[regmod.parameter.Parameter]] = None, param_specs: Optional[Dict[str, Dict]] = None)[source]¶

Bases: object

Model class that in charge of gathering all information, fit and predict.

Parameters

  • data (Data) – Data object.

  • params (Optional[List[Parameter]], optional) – A list of parameters. Default to None.

  • param_specs (Optional[Dict[str, Dict]], optional) – Dictionary of all parameter specifications. Default to None.

Raises

  • ValueError – Raised when both params and param_specs are None.

  • ValueError – Raised when data object is empty.
